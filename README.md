# Instagram App MERN
SocialApp-MERN 🚀

A full-stack social media application built with the MERN stack (MongoDB, Express.js, React.js, Node.js).

📌 About The Project

SocialApp-MERN is a social media platform where users can create accounts, manage profiles, share posts, interact with other users, and communicate through comments and follows.

The project is built with a modern full-stack architecture using React for the frontend and Node.js/Express for the backend.

⚙️ Installation & Setup

1. Clone Repository

git clone https://github.com/AmirNaji3292/SocialApp-Mern.git

Enter the project:

cd SocialApp-Mern

⸻

Backend Setup

Go to backend folder:

cd backend

Install dependencies:

npm install

Create a .env file inside the backend folder:

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key

Run backend server:

npm start

⸻

Frontend Setup

Open another terminal:

cd frontend

Install dependencies:

npm install

Run React application:

npm start

⸻

🌐 API

Backend runs on:

http://localhost:5000

Frontend runs on:

http://localhost:3000
